# Lnkbase Features Bank

Reference bank of Chatbase feature updates that may be useful when planning Lnkbase. This list excludes new model announcements and focuses on platform, channel, support, automation, analytics, and enterprise capabilities.

## Major Platform Features

### Outbound WhatsApp Campaigns

Chatbase supports proactive WhatsApp campaigns, letting teams send personalized updates, reminders, promotions, and follow-ups through AI agents using approved Meta templates.

### HIPAA Compliance

Enterprise customers can use Chatbase in HIPAA-compliant environments after signing a BAA. Chatbase enables Zero Data Retention and disables non-HIPAA-compatible features automatically.

### Chatbase Widgets

AI agents can respond with rich interactive UI components, such as order summaries, flight details, product cards, booking confirmations, weather cards, and custom structured interfaces.

### Chatbase Help Desk

A centralized workspace for managing customer conversations that need human support. AI can draft responses, while human agents review, approve, assign, or take over tickets.

### Chatbase Voice

AI agents can handle inbound phone calls, enabling voice-based customer support using the same agent deployed across chat, email, messaging, and voice.

### Conversation Attachments

Users can attach PDFs and images in supported channels including chat bubble, email, WhatsApp, Instagram DM, and Facebook Messenger.

## Channels And Integrations

### Email Support

Chatbase agents can connect to a support inbox, read incoming emails, and send replies using the agent's knowledge base.

### Native Shopify Integration

Stores can deploy Chatbase easily and let customers check orders, view carts, update billing details, get recommendations, and take shopping actions in chat.

### Stripe Integration And Actions

Customers can manage billing, subscriptions, invoices, billing addresses, and contact info directly inside the chat.

### Zendesk And Salesforce Tickets As Sources

Agents can learn from historical support tickets from Zendesk and Salesforce.

### Zendesk Integration

Agents can create Zendesk tickets and hand off conversations to human agents through Zendesk Sunshine.

### Instagram, Messenger, And WhatsApp Integrations

Chatbase added or improved integrations for Instagram DM, Facebook Messenger, and WhatsApp.

### Zapier Lead Trigger

Lead submissions from Chatbase can trigger Zapier workflows and send data to CRMs or other tools.

## Agent Intelligence And Knowledge Management

### Detect Conflicting Information In Sources

Chatbase can identify outdated or contradictory information in training sources and suggest fixes.

### Suggestions For Missing Sources

Agents can detect unanswered customer questions caused by knowledge gaps and suggest content to add.

### Faster And More Reliable Training

The URL training pipeline was rebuilt for better extraction, faster processing, and improved handling of complex websites.

### See What Your Agent Is Trained On

Users can inspect the crawled source content used for training.

### Search And Sort Sources

Source management includes better search and sorting tools.

### Website URL Wildcards

Users can include or exclude specific URL paths more precisely when crawling websites.

### Agent Size Limit Increased

Training capacity increased from 11 million to 33 million characters.

## UI, Conversation, And Customer Experience

### Agent Page

A full-page chat-style help page where customers can ask questions instead of browsing documentation manually.

### Nested Suggested Messages

Suggested messages can be grouped under parent options, such as "Orders -> Track order / View order / Update billing details."

### Dictation

Customers can speak instead of typing across supported channels, including website widget, agent page, and Meta channels.

### Localized Chat UI

Chat menus, modals, and tooltips can be localized in 40+ languages, including right-to-left languages.

### Fresh New AI Agent Design

Chatbase updated the look of AI agents with cleaner layouts and more modern message bubbles.

### Client-Side Custom Forms

Teams can build custom forms inside the chat experience for leads, surveys, feedback, or workflows.

### Bug Capture Modes

Support widgets can offer bug-reporting capture modes such as screenshots, desktop recordings with microphone state, instant replay, and shareable recording request links. This gives support teams richer evidence than a plain text ticket.

### Smart Follow-Ups

Agents can guide users with clickable follow-up options based on conversation context and common questions.

### Multi-Conversation Support

Users can manage multiple chats, close conversations, and revisit recent chats.

### Custom Chatbot Footer

Chat widgets can include custom footer text or links, such as privacy policies or disclaimers.

### Pattern Summary

The most requested items are mostly about bigger platform capabilities: integrations, multi-product support, roadmap/feedback workflows, support channels, analytics, and Help Center/changelog control.

The newest requests are much more operational. They focus on polish and workflow depth: ticket management, Fibi behavior, Messenger edge cases, export tools, SEO controls, workflow conditions, and better admin safety.

Overall, Featurebase users seem to be asking for two things at once: more enterprise-grade integrations and reporting, plus more control over daily support/ticket workflows.
